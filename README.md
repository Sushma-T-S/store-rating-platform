# Store Rating Platform

A Full Stack web application that allows users to register, log in, rate stores, and enables administrators to manage users and stores.

---

## Tech Stack

### Frontend
- React.js
- TypeScript
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- TypeScript
- JWT Authentication
- bcryptjs

### Database
- PostgreSQL

### ORM
- Prisma ORM

### API Testing
- Postman

---

## Features

### Authentication
- User Registration
- Login
- JWT Authentication
- Role-Based Authorization

### Admin
- Dashboard
- Total Users
- Total Stores
- Total Ratings
- Add User
- Add Store
- Filter Users
- View User Details
- View Stores

### User
- Register
- Login
- Search Stores
- Submit Rating
- Update Rating
- Logout

### Store Owner
- Login
- View Store Rating
- View Users Who Rated Store
- Logout

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Install frontend

```bash
cd frontend
npm install
npm run dev
```

Install backend

```bash
cd backend
npm install
npm run dev
```
